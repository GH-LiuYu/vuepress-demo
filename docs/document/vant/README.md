## 安装vant

脚手架

````
# 安装 Vue Cli
npm install -g @vue/cli

# 创建一个项目
vue create hello-world

# 接着在现有项目中使用 Vant 时，可以通过 npm 进行安装：

# Vue 2 项目，安装 Vant 2
npm i vant@2

# Vue 3 项目，安装 Vant 3
npm i vant@3

````
